# Changelog

## [0.1.0] - Initial Release
- **Reset 3D View**: Operator to reset viewport location and rotation.
- **Silhouette Toggle**: Operator to toggle silhouette shading mode.
- **Set Dimensions**: Operator to set absolute world dimensions for selections in Edit Mode.
- **Smart Delete**: Context-aware delete operator (Verts/Edges/Faces).
- **Quick Shelf**: Added Primitives, Lights, and Quick Materials toolbar.
- **Quick Materials**: Operator to create and assign basic PBR materials.
- **UI**: Added "BFA Tools" Sidebar panel.
- **Integration**: Added menu entries to View and Mesh menus.
- **Preferences**: Added toggles for menus and keymaps.
